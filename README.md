# This is a simple IPv4 checker for blacklists.

## How to use

### 1. Clone the repo

   ```bash
   git clone https://github.com/Kasper886/IP-Checker.git
   cd IP-Checker
   ```

### 2. Install dnspython

  ```bash
  py -m pip install dnspython
  ```

### 3. Create file networks.txt
   Put each subnet in separate line (see example)

### 4. Run script

   ```bash
   py IPChecker.py
   ```

### 5. In few minutes you will see the csv file with subnetes, listed in spam list.
